# MTG Japanese Frame Card Maker

## Run
```bash
npm install
npm run dev
```

## Replace assets with your real ones
- `src/assets/mana/*` : replace SVG placeholders by your PNGs (or keep SVG)
- `src/assets/textures/parchment.svg` : replace by your texture
- `src/assets/frame/jp-frame.svg` : replace by your overlay frame
- `src/assets/frame/hanko.svg` : replace by your seal
